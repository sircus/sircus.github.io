Download on GitHub

Install:

  • postcss-import
  • postcss-custom-properties
  • postcss-custom-media
  • postcss-calc
  • postcss-color-function
install PostCss Plugin
$ npm install --save-dev postcss-import postcss-custom-properties postcss-custom-media postcss-calc postcss-color-function
# or
$ npm install --save-dev postcss-import postcss-cssnext
install sircus
$ npm install sircus
CSS:
@import "sircus";

You can import individually by using the postcss plugins and npm.

e.g:

install sircus modules
$ npm install --save-dev sircus-variables sircus-components-flex sircus-components-display sircus-tools-width

@import "sircus-components-flex";
@import "sircus-tools-display";
@import "sircus-tools-display/responsive";
@import "sircus-tools-width";
@import "sircus-tools-width/responsive";
@import "sircus-variables";
